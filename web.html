Questions et Réponses :

  • Que désigne « Le Web » ?
    C’est un système permettant de consulter, à l’aide d’un navigateur, des pages accessibles sur des sites via Internet.
  • Quelle est la condition essentielle pour votre ordinateur, pour que vous puissiez accéder au Web ?
    Il doit être connecté à Internet.
  • Où sont stockées toutes les ressources du Web ?
    Elles sont stockées sur des serveurs partout dans le monde.
  • Expliquer le processus des échanges client-serveur.
    Le client (votre navigateur) envoie une requête au serveur pour demander une page ; le serveur renvoie la page demandée.
  • Nommer des protocoles qui régissent les échanges entre le client et le serveur.
    HTTP, HTTPS (version sécurisée de HTTP).
  • Qu’est-ce qu’une URL ? De quoi est-elle composée ?
    Une URL (Uniform Resource Locator) est l’adresse d’une ressource sur le Web. Exemple : https://www.example.com/index.html contient : - le protocole (https), - le nom de domaine (www.example.com), - le chemin du fichier (/index.html).
  • Qu’est-ce qu’un moteur de recherche ?
    C’est un site permettant de trouver d’autres pages Web à partir de mots-clés (exemples : Google, Bing, Qwant).
  • Expliquer l’usage et le fonctionnement des robots Crawler ou Spider.
    Ce sont des programmes qui parcourent automatiquement les pages Web pour les indexer dans les moteurs de recherche.
  • Expliquer comment un moteur de recherche propose des URL à partir de mots clés.
    Il compare les mots-clés de l’utilisateur avec son index, puis affiche les pages les plus pertinentes.
  • Qu’est-ce qu’une page Web ? Que peut-elle contenir ?
    Une page Web est un document consultable via un navigateur. Elle peut contenir du texte, des images, des vidéos, des liens, etc.
  • Nommer un langage pour créer une page Web.
    HTML (HyperText Markup Language).
  • Comment appelle-t-on le système utilisé par le langage HTML ?
    Le système de balises.
  • Qu’est-ce que le CSS ?
    CSS (Cascading Style Sheets) est un langage qui permet de mettre en forme une page HTML (couleurs, marges, polices…).
  • Est-ce que la réutilisation d’une page Web est toujours autorisée ? Nommer un modèle de licence.
    Non, il faut respecter les droits d’auteur. Exemple de licence : Creative Commons.
